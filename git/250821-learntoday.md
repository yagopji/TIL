### Command (Important!! Don't forget!!)

1. Shell command

```shell
$ pwd : 현재위치
$ ls :파일 목록
$ ls -a : 파일목록(숨김파일포함)
$ ls -l : 상세정보
$ ls -al : 상세정보(숨김파일포함)
$ mkdir : 폴더 만들기
$ touch : 파일 만들기
$ mv : 파일이동, 이름변경
$ cp : 복사
$ rm : 삭제
$ cd : 위치이동
$ cat : 파일보기
```

2. Vim command

normal mode

```text
i - insert mode
ESC - back to normal mode
```
Cmd-line mode
```text
:q - quit
:q! - oeverride and quit
:w - write
:wq - write and quit
:{num} - Go to {num}th line
```

3. Git command

```shell
$ git vi (file name)
$ git add (file name)
$ git commit
$ git push origin main
$ git gitignore
$ git status
$ git clone
$ git branch (branch name)
$ git switch (branch name)
$ git merge (branch name)
```
